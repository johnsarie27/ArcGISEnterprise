# ConvertFrom-IISLog

## SYNOPSIS
Convert from IIS log

## SYNTAX

```
ConvertFrom-IISLog [-Path] <String> [<CommonParameters>]
```

## DESCRIPTION
Convert from IIS log to objects

## EXAMPLES

### EXAMPLE 1
```
ConvertFrom-IISLog -Path C:\logs\myLog.log
Converts log data into objects
```

## PARAMETERS
