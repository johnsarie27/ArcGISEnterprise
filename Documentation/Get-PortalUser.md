# Get-PortalUser

## SYNOPSIS
Get Portal user

## SYNTAX

```
Get-PortalUser [-Context] <Uri> [-Username] <String> [-Token] <String> [<CommonParameters>]
```

## DESCRIPTION
Get Portal user

## EXAMPLES

### EXAMPLE 1
```
Get-PortalUser -Context 'https://arcgis.com/arcgis' -Username 'joe' -Token $token
Get user 'joe' from Portal
```
